  • The Mastery of pandas Part II

DataFrame Attributes

  • head(n): 返回前n行或后n行数据，n默认5
  • info(): 返回概要信息（多少行，多少列，每列有多少非空数据，列的数据类型）
  • describe(): 返回一些统计信息（均值、方差、极值等）

Methods

pd.read_csv()

  • sep=’|’: 设置分隔符
  • names=[‘’, ‘’]: 设置列名

索引

  • df[:10] - 前10行
  • df.loc[1] - index=1的行
  • df.iloc[1] - 第2行(0为第一行)
  • df[‘sex’] - sex 列
  • df.iloc[0:5][‘Sex’] - 前5行的 sex 列
  • df.loc[[‘Braund, Mr. Owen Harris’,’Cumings, Mrs. John Bradley (Florence Briggs Thayer)’],[‘Survived’, ‘Pclass’, ‘Sex’]] - 左行标签，右列表前
  • df = df.drop([‘Parch’, ‘SibSp’], axis = 1) - 删除两列
  • df = df.set_index(“Name”) - 将 Name 列设置为索引
  • df[‘Cabin’].unique() - 独特值
  • df[‘Cabin’].nunique() - 独特值的数量
  • df[‘Sex’].value_counts() - 独特值及其数量

多表操作

  • df2 = pd.concat([df, new_df], axis=0) - 按行拼接数据表
  • df2[‘full_name’] = df2.apply(lambda row: str(row[‘Name’] + “ “ +row[‘Surname’]),axis=1) - 增加新列
  • merged_df = df.merge(df2, how = ‘left’, left_on=’Name’, right_on=’full_name’).drop(‘full_name’, axis = 1) - 按字段拼接数据表
  • class_groupby = merged_df.groupby(‘Pclass’) - 分组
    • class_groupby[‘Fare’].mean() - 均值

对表进行批量操作

  • merged_df[‘old_or_not’] = merged_df[‘Age’].map(lambda age: 1 if age >=45 else 0)
  • merged_df[‘boolean_sex’] = merged_df[‘Sex’].map({‘male’ : 0, ‘female’: 1})
  • def dolar_to_euro(dollars, exchange_rate):
    euros = dollars*exchange_rate
    return euros
    merged_df[‘Fare_euros’] = merged_df[‘Fare’].apply(dolar_to_euro,exchange_rate = 0.9)
  • merged_df[‘old_woman’] = merged_df.apply(lambda row: row[‘boolean_sex’] and row[‘old_or_not’], axis = 1)
  • old_woman_df = merged_df[merged_df[‘old_woman’] == 1]
  • part_cols = [‘Screw’, ‘Nut’, ‘Nail’, ‘Bearing’]
    duration_part_df = pd.DataFrame(np.random.randint(0,100,size=(100, 4)), columns=part_cols)
  • duration_part_df= duration_part_df.applymap(lambda days: days*24)
